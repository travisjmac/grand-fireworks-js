Grand
Fireworks

A reusable fireworks class with graceful endings, performance presets, backgrounds, contained displays, a Sovereign Crown finale, and glittering multi-line text fireworks.

Quick start

<script src="GrandFireworks.js"></script>
<script>
  const fireworks = new GrandFireworks();
  fireworks.start();
</script>

The zero-argument constructor creates a transparent fullscreen overlay, uses the medium adaptive preset, does not start automatically, and runs indefinitely once started.

Explicit show values override preset values. maxRockets is always clamped to the agreed safety maximum of 25.

Ordinary rockets launch within the middle 55% of the display and travel at a random angle of up to 14° left or right. angleStrength controls how pronounced the fan is. Text rockets remain vertical by default, while the Sovereign Crown finale automatically uses a wider 18° fan.

renderer.preferred: 'auto' uses WebGL for fullscreen shows and reliable Canvas 2D rendering for contained shows. Explicitly use preferred: 'webgl2' with preserveDrawingBuffer: true only when you have tested WebGL in your particular nested layout.

Graceful stopping

await fireworks.stop();
// Stops new launches, finishes active fireworks,
// plays the finale when configured, fades out, then resolves.

fireworks.stop({ immediate: true }); // Emergency stop
fireworks.stop({ finale: false });   // Skip one configured finale
fireworks.finalize();                // Graceful stop with finale

Text fireworks

fireworks.launchText('I love you very much. You are the best.', {
  renderMode: 'hybrid',
  maxLines: 3,
  colors: ['#FFFFFF', '#FF69B4', '#FFD700'],
  holdDuration: 2000,
  dissolveDuration: 1000
});

Text wraps at word boundaries. Each line receives its own rocket; launch times are coordinated so the lines explode into one centered message. Overflow can be ellipsis, truncate, or scale.

Methods

launch(), launchText(), and launchFinale() work even when the regular show is not running. They temporarily wake the renderer, play only the requested effect, and fade away when it finishes—without restarting automatic launches.

MethodPurpose
start(options?)Start the show. A run-specific duration may be supplied.
stop(options?)Gracefully finish and return a Promise.
pause() / resume()Pause or resume without launching a new show.
launch(options?)Launch one regular firework, including while stopped.
launchText(text, options?)Launch synchronized text rockets, including while stopped.
launchFinale()Launch the standalone staged ten-trail Super Grand Finale.
finalize()Gracefully end with the finale.
clear()Remove active fireworks.
setOptions(partial)Merge new settings.
getOptions()Return resolved configuration.
getStats()Return renderer, FPS, particles, rockets, state, and timer.
destroy()Remove layers, listeners, renderer, and restored container styles.

Events

start, pause, resume, stop, textlaunch, and rendererchange. The renderer-change event reports automatic WebGL-to-Canvas fallback.
