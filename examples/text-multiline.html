Several Synchronized Rockets

The message wraps across separate rockets that burst into one centered block.

Configuration

fireworks.launchText(
  'I love you very much. You are the best.',
  {
    renderMode: 'hybrid',
    maxCharactersPerLine: 18,
    maxLines: 3,
    colors: ['#FFFFFF', '#FFD700', '#7FDBFF'],
    revealDuration: 300,
    holdDuration: 2200,
    dissolveDuration: 1100,
    fallDuration: 3600
  }
);
