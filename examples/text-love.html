Text Firework

Press the button to send a glittering message into the sky.

Configuration

const fireworks = new GrandFireworks({
  autoStart: true,
  show: { intensity: 0.55 },
  textFirework: {
    renderMode: 'hybrid',
    colors: ['#FFFFFF', '#FF69B4', '#FFD700'],
    holdDuration: 1800,
    dissolveDuration: 1000,
    exclusive: true
  }
});

fireworks.launchText('I love you');
